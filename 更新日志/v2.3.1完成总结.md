## ⚙️ 配置选项

### 默认设置（推荐）

```python
# 所有功能都开启
analysis = analyzer.analyze_fund(
    fund_code='161725',
    fund_name='招商中证白酒',
    include_flow=True,    # 开启资金流向 ✅
    include_hot=True      # 开启热点分析 ✅
)
```

### 快速模式（可选）

```python
# 只分析均线，速度更快
analysis = analyzer.analyze_fund(
    fund_code='161725',
    fund_name='招商中证白酒',
    include_flow=False,   # 关闭资金流向
    include_hot=False     # 关闭热点分析
)
```

**性能对比**：
- 完整分析：约 3-5 秒
- 快速模式：约 1-2 秒
